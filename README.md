# Hospedaje Las Hermanas — Trabajo Final HTML/CSS

Proyecto académico realizado únicamente con HTML5 y CSS3.

## Estructura

```
mirna-britos-Trabajo HTML/
├── index.html
├── productos.html
├── presupuesto.html
├── contacto.html
├── css/
│   └── styles.css
└── img/ (o Img/ según el sistema)
```

## Requisitos del trabajo (cumplidos)
- HTML5 + CSS3 sin JavaScript
- Menú de navegación en todas las páginas con estado activo
- Portada `index.html` con secciones
- Galería como `productos.html`
- Formulario en `presupuesto.html` (nombre, email, teléfono, textarea, checkbox)
- `contacto.html` con mapa e información
- Aviso legal en el pie de página

## Cómo ejecutar
- Abre `index.html` en tu navegador.

## Publicación en GitHub Pages (resumen)
1. Crea repositorio público en GitHub y sube todos los archivos.
2. En el repo: Settings → Pages → Source: `main` (o `master`) → `/(root)` → Save.
3. Espera a que GitHub genere la URL y ponla arriba.

## Notas
- En servidores como GitHub Pages, las rutas distinguen mayúsculas/minúsculas.
  - Asegúrate de usar `img/` (minúsculas) en carpeta y rutas, o unifica a `Img/` en todas las referencias.
- Icono del sitio (favicon) incluido.
